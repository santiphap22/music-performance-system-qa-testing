# Use Case Specification: รายละเอียดกรณีย้าย Queue เพลง

| Use Case Attribute | Specification Details |
| :--- | :--- |
| **1. Use Case ID** | 9 |
| **2. Use Case Name** | ย้าย Queue เพลง |
| **3. Primary Actor** | Backstage |
| **4. Description** | Backstage สามารถย้าย queue เพลงได้กรณีที่มีการเปลี่ยนแปลงของ queue เพลงก่อนจะเริ่มงานแสดงดนตรี |
| **5. Trigger** | Backstage ต้องการย้าย queue เพลง |
| **6. Preconditions** | PRE-1 Backstage มีการเพิ่มงานแสดงดนตรีลงไปในระบบแล้ว<br> PRE-2 Backstage ได้เพิ่มเพลงลงไปในงานแสดงดนตรีนั้น ๆแล้ว<br> PRE-3 PRE-3 Backstage ทำการยืนยันตัวตนสำเร็จ |
| **7. Postconditions** | POST-1 Backstage ทำการย้าย queue เพลงได้สำเร็จ |
| **11. Priority** | Low |
| **12. Frequency of Use** | 0 – 5 ครั้งต่อหนึ่งงานแสดงดนตรี |

### Workflows & Flow of Events

#### 8. Normal Flow
1. Backstage เลือก "Event" จากหน้า Main Menu 
    - ระบบแสดงรายการงานแสดงดนตรีทั้งหมดที่เกี่ยวข้อง 
2. Backstage เลือกงานงานแสดงดนตรีจากรายการที่มี 
    - ระบบทำการแสดงหน้ารายละเอียดของงานแสดงดนตรีที่เลือก 
3. Backstage กดปุ่ม "ดูรายชื่อเพลง" 
    - ระบบแสดงรายชื่อเพลงที่ใช้ในการแสดงของงานแสดงดนตรีนั้น 
4. Backstage เลือกเพลงที่ต้องการย้าย โดยการคลิกที่เพลงค้างเอาไว้และสามารถเลื่อนได้ตามต้องการ 
5. ระบบแสดงข้อความแจ้งเตือน: "คุณต้องการย้ายลำดับเพลงหรือไม่" พร้อมปุ่ม "ยืนยัน" และ "ยกเลิก" 
6. Backstage กดปุ่มยืนยัน 
    - ระบบจะนำเพลงย้ายจากจุดเดิมไปที่จุดใหม่ 
7. หากต้องการย้ายเพลงอื่น ๆ เพิ่มเติม Backstage ทำซ้ำขั้นตอน 4 - 6 
8. Use Case สิ้นสุดเมื่อ Backstage ไม่ต้องการย้ายเพลงเพิ่มเติม

#### 9. Alternative Flow
**Backstage ยกเลิกการย้าย Queue เพลง**
1. Backstage เลือก "Event" จากหน้า Main Menu 
    - ระบบแสดงรายการงานแสดงดนตรีทั้งหมด 
2. Backstage เลือกงานรายการงานแสดงดนตรีจากรายการที่มี 
    - ระบบทำการแสดงหน้ารายละเอียดของ รายงานแสดงดนตรีที่เลือก 
3. Backstage กดปุ่ม "ดูรายชื่อเพลง" 
    - ระบบแสดงรายชื่อเพลงที่ใช้ในการแสดงของงานแสดงดนตรีนั้น 
4. Backstage เลือกเพลงที่ต้องการย้าย โดยการคลิกที่เพลงค้างเอาไว้และสามารถเลื่อนได้ตามต้องการ 
5. ระบบแสดงข้อความแจ้งเตือน: "คุณต้องการย้ายลำดับเพลงหรือไม่" พร้อมปุ่ม "ยืนยัน" และ "ยกเลิก" 
6. Backstage กดปุ่มยกเลิก 
    - ระบบจะไม่ย้ายเพลงและอยู่ในตำแหน่งเดิมของรายการเพลงใน Event 
7. เพลงยังคงอยู่ในตำแหน่งเดิมของรายการเพลงเดิมโดยไม่มีการเปลี่ยนแปลง 
8. ระบบนำ Backstage กลับไปที่หน้ารายชื่อเพลง 
9. Alternative Flow สิ้นสุด และกลับเข้าสู่ Normal Flow

#### 10. Exceptions
- None
