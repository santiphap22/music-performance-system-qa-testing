# Use Case Specification: รายละเอียดกรณีลบเพลงใน Queue

| Use Case Attribute | Specification Details |
| :--- | :--- |
| **1. Use Case ID** | 10 |
| **2. Use Case Name** | ลบเพลงใน Queue |
| **3. Primary Actor** | Backstage |
| **4. Description** | Backstage สามารถลบเพลงออกจาก Queue ของงานแสดงดนตรีนั้นๆได้ |
| **5. Trigger** | Backstage ต้องการลบเพลงออกจาก Queue เพลงของงานแสดงดนตรีนั้น ๆ เนื่องจากไม่มีการเล่นเพลงนั้น ๆ แล้ว |
| **6. Preconditions** | PRE-1 Backstage ทำการยืนยันตัวตนสำเร็จ<br> PRE-2 จะต้องมีเพลงของถูกเพิ่มอยู่ในระบบก่อนแล้ว |
| **7. Postconditions** | POST-1 เพลงนั้น ๆ ถูกลบออกจากระบบ |
| **11. Priority** | Medium |
| **12. Frequency of Use** | 0-3 ครั้งต่อหนึ่งงานดนตรี |

### Workflows & Flow of Events

#### 8. Normal Flow
1. Backstage เลือก "Event" จากหน้า Main Menu 
    - ระบบแสดงรายการงานแสดงดนตรี 
ทั้งหมด 
2. Backstage เลือกงานงานแสดงดนตรีจากรายการที่มี 
    - ระบบแสดงหน้ารายละเอียดของงานแสดงดนตรีที่เลือก 
3. Backstage กดปุ่ม "ดูรายชื่อเพลง"
    - ระบบแสดงรายชื่อเพลงที่ใช้ในการแสดงของงานแสดงดนตรีนั้น 
4. Backstage เลือกเพลงที่ต้องการลบ 
5. Backstage ทำการกดปุ่ม “ลบเพลง” 
6. ระบบแสดงข้อความแจ้งเตือน: "ต้องการลบเพลงหรือไม่" พร้อมปุ่ม "ยืนยัน" และ "ยกเลิก" 
7. Backstage กดปุ่มยืนยัน 
    - ระบบลบเพลงออกจากรายการเพลงใน Event 
    - ระบบแสดงข้อความแจ้งเตือน: "ลบเพลงสำเร็จ" 
8. หากต้องการลบเพลงอื่น ๆ เพิ่มเติม: 
    - ระบบนำ Backstage กลับไปที่หน้ารายชื่อเพลง 
9. Backstage ทำซ้ำขั้นตอน 4 - 8 
10. Use Case สิ้นสุดเมื่อ Backstage ไม่ต้องการลบเพลงเพิ่มเติม 

#### 9. Alternative Flow
**Backstage ยกเลิกการลบเพลง**
1. Backstage เลือก "Event" จากหน้า Main Menu 
    - ระบบแสดงรายการงานแสดงดนตรี ทั้งหมดที่เกี่ยวข้อง 
2. Backstage เลือกงานงานแสดงดนตรีจากรายการที่มี
    - ระบบแสดงหน้ารายละเอียดของงานแสดงดนตรีที่เลือก 
3. Backstage กดปุ่ม "ดูรายชื่อเพลง" 
    - ระบบแสดงรายชื่อเพลงที่ใช้ในการแสดงของงานแสดงดนตรีนั้น ๆ 
4. Backstage เลือกเพลงที่ต้องการลบ โดยเลื่อนเพลงลงไปที่ถังขยะ 
5. ระบบแสดงข้อความแจ้งเตือน: "ต้องการลบเพลงหรือไม่" พร้อมปุ่ม "ยืนยัน" และ "ยกเลิก" 
6. Backstage กดปุ่มยกเลิก 
7. เพลงยังคงอยู่ในรายการเพลงเดิมโดยไม่มีการเปลี่ยนแปลง 
8. Alternative Flow สิ้นสุด

#### 10. Exceptions
- None
